# Assignment Report Draft

## Objective

Build a working GPT-style language model in my GitHub repository for the Artificial Intelligence and Financial Engineering course.

## Course Progression

The implementation follows six course notebooks:

1. Bigram language model
2. MLP character-level language model
3. MLP on a larger text corpus
4. GPT-style next-token dataset
5. Single-head masked self-attention
6. TinyGPT with multi-head attention and Transformer blocks

## Current Status

The repository includes a trainable TinyGPT implementation, data preparation script, training script, sampling script, and documentation.

## Financial Engineering Extension

The model can be trained on finance-related text to generate domain-shaped language around markets, risk, portfolio management, and derivatives.
